The Labyrinth

Select the interface that you want to use

Through the admin interface menu you can add more information to the map about the points to reach, turns and junctions (create nodes and links).

Through the user interface menu you can set a starting and a final point of your destination, and get the best route.

Through the button below you go back to the home page.

Create paths (Admin Interface)

By clicking the button below you will enter the menu that allows you creating nodes.

By clicking the button below you will enter the menu that allows you creating links.

By clicking the button below you will be taken back to interface option menu.

Manage Nodes

For creating a node click the button below. The nodes represent all the created points in the structure where is a change of direction.

Nodes badly placed can be changed by clicking the button below.

By clicking the button below you will be taken back to admin interface.

Create new node

Drag the red node in a position desired and then save it.

Modify the node

First select the node wanted and then you can modify it.

Manage Links

For creating a link click the button below. The links connect all the nodes created in the structure.

The link of badly connected nodes can be removed by clicking the button below.

By clicking the button below you will be taken back to admin interface.

Create new link

Pressing the "Select the A node" and "Select the B node" buttons allows selecting the nodes you want to connect. After selected the two nodes you have to press the third "?" button for calculating the length of the path between these two nodes. Only when you have done these actions you'll be able to save the link.

Delete a link

First, click the link you want to delete and after it turned red you can remove it easily.

Get indications (User Interface)

Pressing the "Select the starting node" button allows choosing your current location and "Select the ending node" allows choosing your final destination.

After that you have to click on "Calculate Path" button for having the total distance and the path that you have to follow.
