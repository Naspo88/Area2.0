Tutorial 2 of 7:

Before starting to create a map here you can find a definition of graph, node and link:

  • Graph: a structure represented by a diagram consisting of points (nodes) joined by lines (links)
  • Node: is the base of the structure graph of the map, each node created will be depicted as a circle and will indicate a point on the map where there is a possible destination, an intersection between more than one aisle or a point of interest.
  • Links: are the possible ways that you can deal with, and are represented by a line, they must be used to indicate that there is a passage from one node to another.

If you want to know how to create a node, go to the next tutorial

Otherwise close the tutorial

Tutorial 3 of 7:

Let's try to create a point of interest in the map.

Follow the steps below:

  • Click on the "Create paths (Admin Interface)" button.
  • Click on the "Manage Nodes" button.
  • Click on the "Create new node" button.
  • Drag the red circle somewhere in the map, add the label and save by clicking "Save Node" button.

Tutorial 5 of 7:

In this tutorial, we will learn how to create paths that final users will use. By connecting two nodes together you will create a possible path. Let's create some.

Follow the steps below:

  • Click on the "Manage Links" button.
  • Click on the "Create new link" button.
  • Click on the "Select the A node" button. And click to the first node (colored in green) that you want to connect.
  • Click on the "Select the B node" button. And click to the second node (colored in green) that you want to connect.
  • Click on the "Calculate length" button.
  • Now you have create your first link, save it clicking on "Save Link" button.
  • Very good! Now the final user can use this path for going between this two points. Finish the map by connecting the others nodes and create another four links.
  • Excelent! Now you have created a complete map! Before using it let's see how to delete a link. Go to the next tutorial
    Otherwise close the tutorial

Tutorial 6 of 7:

With this tool it is possible to delete paths positioned in error, for example those that pass over a wall. With this tutorial we will see how to do it.

Follow the steps below:

  • Click on the "Manage Links" button.
  • Click on the "Delete a link" button.
  • Click on one of the green lines that you have created. When it becomes red delete it by clicking the "Delete Link" button.
  • Excelent! Now you know how to delete a link! Now that you know everything about creating a map with Area 2.0 tool. Create one and when you are satisfied about your creation let's look how your final user will use it. Go to the last tutorial
    Otherwise close the tutorial

Tutorial 7 of 7:

In the last tutorial of this mini-course we will see how you can use the map, that we have created so far, to find the shortest paths to move in the building. Let's see how to do it.

Follow the steps below:

  • Click on the "Back" button. To return to the selection of the interfaces.
  • Click on the "Get indications (User Interface)" button.
  • Click on the "Select the starting node" button. When circles become green, select the starting point of the path that you want to calculate.
  • Click on the "Select the ending node" button. When circles become green, select the point that you want to achieve with the path.
  • Click on the "Calculate Path" button to display in red the shortest path between the points you've chosen.
  • Excelent! Now you know how to use the area 2.0 to its maximum potential (temporary), you can create any map you need within the site, to create new ones, please contact me through the contact us page. R.

Modify the node

Manage Links

For creating a link click the button below. The links connect all the nodes created in the structure.

The link of badly connected nodes can be removed by clicking the button below.

By clicking the button below you will be taken back to admin interface.

Create new link

Pressing the "Select the A node" and "Select the B node" buttons allows selecting the nodes you want to connect. After selected the two nodes you have to press the third "Calculate Length" button for calculating the length of the path between these two nodes. Only when you have done these actions you'll be able to save the link.

Delete a link

First, click the link you want to delete and after it turned red you can remove it easily.

Get indications (User Interface)

Pressing the "Select the starting node" button allows choosing your current location and "Select the ending node" allows choosing your final destination.

After that you have to click on "Calculate Path" button for having the total distance and the path that you have to follow.
