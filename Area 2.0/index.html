The tool is part of a larger project that wants to propose a new technique to solve the problem of indoor navigation.

  • An Invented Place

    It's a map that envisions a building with offices and rooms. The map is not populated: ideal for understanding how you can create a map and how to use what has been created.

  • The Labyrinth

    The Labyrinth is a structure consisting of intricate paths lined with high hedges and having an exit difficult to find. It is a good example to see how the map created earlier works and how it can be used.
